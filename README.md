### Descrizione
Questa repository contiene una serie di esercizi e piccole applicazioni in Java
che vengono progettati e scritti in classe per consolidare le basi di:
- Definizione di classe ad oggetti;
- Gestione di array e collezioni;
- Incapsulamento, ereditarietà, composizione;
- Logica di base e manipolazioni dati.
È utile come supporto didattico per studenti di informatica o come raccolta di esempi
per chi vuole ripassare concetti fondamentali di Java
### Struttura della Repository
La repository è organizzata in questo modo:
- Cartelle contenenti i codici sorgente dei progetti
- File di configurazione Git, come .gitignore
- README.md
### Come usare i progetti
1. Clona la repository
   `git clone https://github.com/TomRiddle901/Java.git`
2. Apri il progetto (o la directory dell'esercizio) con il tuo IDE Java predefinito
   (NetBeans, IntelliJ IDEA, Eclipse, VS Code + Estensione Java - consigliato solo se non hai altri IDE, perché alcune funzionalità avanzate potrebbero non essere disponibili)
3. Esegui il programma main (di solito si chiama MainProgram.java)
   oppure compila con `javac`
### Esempi di esecuzione
Supponendo di voler eseguire il programma *Esercizio_classe_calciatore*
da terminale potresti fare:
<pre>```cd Java/Esercizio_classe_calciatore
        javac *.java
        java Main```
</pre>
(Sostituisci *Main* con il nome della classe con metodo *main* corretta presente nel progetto)
